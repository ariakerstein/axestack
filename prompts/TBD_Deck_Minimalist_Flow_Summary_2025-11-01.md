## 🎯 Use Cases for Each Mode

### Minimalist Mode (For Live Pitch)
- **When:** Live investor meetings, stage presentations
- **Why:** Maximum emotional impact, speaker controls narrative
- **Experience:** VC nods, doesn't read
- **Outcome:** Investors feel, not just understand

### Detailed Mode (For Leave-Behind / Email)

- **Experience:** Can be understood without speaker
