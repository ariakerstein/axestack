## 📋 SOURCE CITATION FORMAT

For each source, document:
```
[Claim]
Source: [Full citation]
Date: [Publication date]
URL/DOI: [If available]
Key stat: [Exact quote from source]
Context: [How we're using it]
```

**Example:**
```
1.9M cancer diagnoses yearly
Source: American Cancer Society. Cancer Facts & Figures 2024.
Date: January 2024
URL: https://www.cancer.org/research/cancer-facts-statistics/all-cancer-facts-figures/2024.html
Key stat: "An estimated 1.9 million new cancer cases are expected to be diagnosed in the United States in 2024."
Context: Used to quantify market size and problem scale
```
